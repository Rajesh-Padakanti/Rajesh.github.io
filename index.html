Title: Implementation IPv6 VPN services (6VPE) OVER MPLS

Description: This project work is about implementation of IPV6 in MPLS(MULTIPROTOCOL LABEL SWITCHING) network.As the IPV6 has been deployed in the internet core networks and many content providers provide service using the new protocol,various internet service providers are lifted behind due to the high cost of migration especially for MPLS core.
